# Harbor Market

シンプルな EC デモサイトです。静的な HTML/CSS/JavaScript のみで構成されており、検索やカテゴリフィルタ、価格スライダー、人気商品の絞り込み、ショッピングカート機能（ローカル保存付き）などを体験できます。

## 使い方

1. リポジトリをクローンするか ZIP をダウンロードして展開します。
2. `index.html` をブラウザで開くだけでサイトを表示できます。
3. 商品をカートに追加すると、右側のカートパネルが開き、合計金額や数量が更新されます。カートの内容はブラウザの `localStorage` に保存されるため、ページを再読み込みしても保持されます。

## ローカルでのプレビュー

開発用に簡易サーバーを立ち上げる場合は、Python がインストールされていれば以下のコマンドで確認できます。

```bash
python -m http.server 5173
```

ブラウザで `http://localhost:5173` にアクセスするとサイトを表示できます。

## 注意事項

- 本サイトはデモ目的のため、決済処理や注文確定などの機能は実装されていません。
- 画像は Unsplash の外部 CDN を参照しています。インターネット接続が必要です。
- カート内容はブラウザに保存されるのみで、サーバーには送信されません。
